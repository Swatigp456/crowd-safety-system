Real-Time Alert API Documentation

Base URL: http://127.0.0.1:8000/api/
1. Get Alerts
GET /api/alerts/
Response: 
{
  "status": "success",
  "count": 5,
  "alerts": [
    {
      "id": 1,
      "title": "High Crowd Density",
      "message": "Times Square is very crowded",
      "priority": "high",
      "location": "Times Square",
      "latitude": 40.7128,
      "longitude": -74.0060,
      "created_at": "2024-03-27 15:30:00",
      "read": false
    }
  ]
}
2. Send Alert (Admin/Security Only)
POST /api/alerts/
Request Body: 
{
  "title": "Alert Title",
  "message": "Alert message",
  "alert_type": "security",
  "priority": "high",
  "location": "Times Square",
  "latitude": 40.7128,
  "longitude": -74.0060,
  "send_to_all": true
}
Response: 
{
  "status": "success",
  "alert_id": 123,
  "message": "Alert sent successfully"
}
3. Trigger Panic
POST /api/panic/
Request Body: 
{
  "latitude": 40.7128,
  "longitude": -74.0060,
  "message": "Emergency!",
  "location": "Times Square"
}
Response: 
{
  "status": "success",
  "panic_id": 45,
  "alert_id": 124,
  "message": "Emergency alert sent!"
}
4. Acknowledge Alert
POST /api/alerts/<alert_id>/acknowledge/
Response: 
{
  "status": "success",
  "message": "Alert acknowledged"
}
5. Get Crowd Data
GET /api/crowd/
Response: 
{
  "status": "success",
  "count": 10,
  "data": [
    {
      "location": "Times Square",
      "latitude": 40.7128,
      "longitude": -74.0060,
      "density": "high",
      "count": 847,
      "timestamp": "2024-03-27 15:30:00"
    }
  ]
}
